## Our Services

### Two Diagnostic Tools

#### 1. AI Readiness Scanner (Automated)

Technical scan of a URL across 7 dimensions. Available free at crawlready.dev.

| Check | What It Measures |
|---|---|
| llms.txt | LLM communication file |
| robots.txt | AI crawler access |
| Structured Data | JSON-LD / Schema.org markup |
| Meta Tags & Open Graph | Title, description, OG tags |
| Sitemap | Page discoverability |
| HTTPS | Security baseline |
| Response Time | Speed for AI crawlers |

**Output**: Score 0-100, Grade A-F. Useful as a quick health check and lead magnet.

#### 2. Agentic Test (Manual, High-Value)

We give a real AI agent a real task — the kind a potential customer would ask:

> "I need an SEO agency in Madrid. Find one and request a quote."

Then we document step-by-step what happens:
- Can the AI find the business?
- Can it understand the services and pricing?
- Can it fill out a contact form or request a quote?
- Can it use WhatsApp, chat, or email?
- What breaks? What's invisible?

**Output**: Visual HTML report with pass/fail for each step, technical details, competitor comparison, and specific recommendations. This is our core differentiator — nobody else does this.
